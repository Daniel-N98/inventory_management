## Features
- Creating Team Members
  - Team roles (Employee, Editor, Admin)
    Team roles allow Managers/Admins to control access to certain features within the application.
- Creating Inventory Items
  Inventory items can be viewed, created, updated, and deleted. Access is controlled by Team Roles.
- Creating Categories
  Categories can be viewed, created, updated, and delated.

Tables are populated with the following: Team members, inventory items, and Categories.
